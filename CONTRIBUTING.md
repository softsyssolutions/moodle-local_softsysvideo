# Contributing

## Development setup

1. Clone this repository into `{moodle_root}/local/softsysvideo/`
2. Enable Moodle developer mode: `Site administration → Development → Debugging`

## Coding standards

This plugin follows [Moodle Coding Style](https://moodledev.io/general/development/policies/codingstyle).

## AMD modules

JavaScript lives in `amd/src/`. After changes, copy to `amd/build/` (or run grunt).

## Pull requests

- Follow Moodle coding style
- Include meaningful commit messages
- Update CHANGELOG.md
